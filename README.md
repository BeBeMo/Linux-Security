# Linux-Security

**_项目简述_**：在Linux下对linux安全管家软件的功能进行优化和加强，主要用到qt开发环境，c++、HTML5、CSS3、JavaScript、linux shell等编程语言。  
**_功能描述_**：包括一键体检、垃圾清理、防火墙、病毒查杀、优化加速、基本信息、进程管理、用户管理等功能。
