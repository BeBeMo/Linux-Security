# Linux-Security

**_项目简述_**：在Linux下用node-webkit环境开发linux安全管家软件，主要用到HTML5、CSS3、JavaScript、Linux shell等编程语言。  
**_功能描述_**：包括一键体检、垃圾清理、防火墙、病毒查杀、优化加速、文件备份、数据加密、安全保险箱等功能。
